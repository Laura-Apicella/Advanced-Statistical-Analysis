# Advanced Statistical Analysis

## Overview

This project showcases the application of advanced statistical techniques to solve different analytical problems using real-world datasets and SAS.

The analyses include:

* Multiple Linear Regression for housing price prediction
* Logistic Regression for Titanic survival analysis
* Exploratory Factor Analysis (EFA) for latent factor identification

## Project Contents

### 1. Housing Price Prediction – Multiple Linear Regression

Developed and optimized a multiple linear regression model to explain housing prices based on property characteristics such as taxes, living area, lot size, and location.

Key techniques:

* Pearson correlation analysis
* Multicollinearity assessment
* Backward variable selection
* Model interpretation and validation

### 2. Titanic Survival Analysis – Logistic Regression

Built a logistic regression model to identify the factors influencing passenger survival on the Titanic.

Key techniques:

* Binary outcome modeling
* Odds Ratio interpretation
* Likelihood Ratio, Score and Wald tests
* Model fit assessment

### 3. Exploratory Factor Analysis

Performed exploratory factor analysis on survey data regarding patients' perceptions of their primary care physicians.

Key techniques:

* Maximum Likelihood Factor Analysis
* Kaiser criterion
* Scree Plot analysis
* Varimax rotation
* Latent factor interpretation
